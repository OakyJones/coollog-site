# CoolLog — Landingsside

B2B landingsside for CoolLog, bygget med Next.js 14 + Tailwind CSS.

## Kom i gang

```bash
cd coollog-site
npm install
npm run dev
```

Åbn [http://localhost:3000](http://localhost:3000) i din browser.

## Deployment

```bash
npm run build
npm start
```

Eller deploy til Hetzner med Docker — se `next.config.js`.
